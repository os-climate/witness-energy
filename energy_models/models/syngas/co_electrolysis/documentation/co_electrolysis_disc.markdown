# CoElectrolysis

High temperature co-electrolysis is a process which electro-chemically reduces H2O and CO2 using solid oxide electrolyzer cell via the reactions in the equations :

$$H_2O -> H2 + 1/2 O_2$$

$$CO_2 -> CO + 1/2 O_2$$

Total reaction is :

$$H_2O + CO_2 -> H2 + CO + O_2$$
